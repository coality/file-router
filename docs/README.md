# FileRouter — Documentation

The FileRouter technical specification is available in two languages, kept in sync.

| Language | Index |
|----------|-------|
| 🇫🇷 Français | [`fr/README.md`](fr/README.md) |
| 🇬🇧 English | [`en/README.md`](en/README.md) |

Shared, language-neutral artifacts:

- **JSON Schemas**: [`schemas/`](schemas/) — `metadata.schema.json`,
  `audit.schema.json`, `config.schema.json`
- **Reference examples**: [`examples/`](examples/) — config, metadata and audit
  instances (validated against the schemas)

> La spécification est maintenue en français (`fr/`) et en anglais (`en/`). Toute
> évolution doit être répercutée dans les deux langues.
